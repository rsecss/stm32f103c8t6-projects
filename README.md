# STM32F103C8T6 Projects

## Introduction

本仓库主要是存储以 STM32F103C8T6 为主控芯片拓展开的工程文件，包含了大部分的 STM32 入门工程文件，如果需要相关的工程模版和参考资料，可前往查看我的另一个仓库[stm32-template](https://github.com/rsecss/stm32-template)
